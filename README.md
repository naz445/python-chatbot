# python-chatbot

A simple rule-based Python chatbot that responds to user input using conditional statements.  
It demonstrates basic concepts of interactive programming, string matching, and control flow in Python.

## Features
- User-friendly conversational loop
- Responds based on detected keywords
- Predefined rule-based answer system
- Clean and easy-to-read script structure

## Technologies Used
- Python 3

## What It Can Do
- Recognize greetings (e.g., "hello", "hi")
- Answer simple questions
- Provide basic informational responses
- End the chat when the user types "bye"

## Purpose
This project was built for practice and showcases essential Python skills such as loops, functions, conditions, and basic text processing.
